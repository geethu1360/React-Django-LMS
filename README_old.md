# Apponix LMS — Learning Management System

Apponix LMS is a comprehensive, full-stack learning platform designed for managing courses, tracking student progress, and handling digital enrollments. Built with a robust **Django REST Framework** backend and a dynamic **React** frontend, it mimics features found in professional platforms like Udemy.

## 🚀 Features
- **Course Catalog:** Browsable listing of courses including "Hotel Management using Django".
- **Student Dashboard:** Track enrolled courses and lesson completion.
- **Advanced Cart System:** Add courses to cart, apply coupons, and manage orders.
- **Detailed Course Views:** Explore modules, lessons, variants (pricing), and Q&A.
- **Secure Authentication:** JWT-based login and registration.
- **Admin Control:** Full management of teachers, country-based pricing, and notifications.

## 🛠 Tech Stack
### Frontend
* **React.js** (Vite-powered)
* **Axios** (API handling)
* **Bootstrap** (UI/UX)

### Backend
* **Django** & **Django REST Framework**
* **SQLite/PostgreSQL** (Database)
* **Cloudinary/Local** (Image & File uploads)
